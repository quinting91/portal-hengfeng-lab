# portal-hengfeng-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。所有页面均为静态内容，不依赖后端服务，可直接通过浏览器访问。

## 项目简介

本仓库主要存放一系列独立的 HTML 文件，涵盖工具页面、信息展示页、实验性页面等。每个页面均为自包含的单一文件，便于分发、存档和快速部署。

## 目录结构

```
.
├── pages/           # 存放所有 HTML 页面文件
├── assets/          # 共享资源（如 CSS、图片、字体等）
├── README.md        # 本文件
└── .gitignore       # Git 忽略规则
```

各页面按功能或用途命名，放置在 `pages/` 目录下。资源文件统一存放在 `assets/` 中，以便复用。

## 页面归档说明

- 所有页面均为纯前端实现，无服务端依赖。
- 页面内可能包含外部资源引用（如 CDN 库），但不会收集用户数据。
- 归档的页面可能不定期更新或移除，以保持仓库内容整洁。

## 维护说明

- 欢迎提交 Issue 报告页面问题或提出建议。
- 如需贡献新页面，请确保文件为自包含 HTML，并放置在 `pages/` 下。
- 提交前请检查文件是否符合基本 HTML 规范，且不包含恶意代码或隐私追踪功能。
- 本仓库不接受涉及商业推广、广告或敏感内容的页面。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/portal-hengfeng-lab.git
   ```
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件即可查看。
